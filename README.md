# CL — Couple's Lifestyle App

Mobile-friendly web app: home, food, movies, 8-ball, trips, op-eds, books, recipes.

**Path:** `Documents\CL` · HTML/CSS/JS · `localStorage` · pink + soft green

## Open

Double-click `index.html`, or:

```powershell
cd $env:USERPROFILE\Documents\CL
npx --yes serve .
```

## Features

| Area | Details |
|------|---------|
| **Home** | Dashboard, names, quick links |
| **Food** | GPS distances, filters, visits, **Ask Grok** |
| **Movies** | Watched / wishlist / recs + **Ask Grok** |
| **Games** | Golf, Darts (501), Spades, Gin Rummy · scores + history |
| **8-Ball** | Under More · Magic 8-Ball |
| **Trips** | NL plan (“Plan a weekend in Paris”), stay/eat/drink/do + **Ask Grok** |
| **Recipes** | Full ingredients & steps for meal + drink + **Ask Grok** |
| **Books** | Shelves + **Ask Grok** |
| **Profile** | Couple photo/names + **xAI API key** |

## Grok

1. Get a key at [console.x.ai](https://console.x.ai/)
2. **Profile → xAI API key** → save
3. Use **Ask Grok** on Food, Movies, Trips, Recipes, Books

Without a key, smart offline mode still works.
